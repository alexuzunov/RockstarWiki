The following is a list of Rockstar Wiki Rules and Policies

Users who break the rules will be warned, and if they continue this path, they will be blocked.
  • Copyright - Do not copy articles form other wikis and add them here. Anyone who copies from other wikis will be warned and if continued, the user will be blocked from editing.
  • Good faith - Every user must edit properly.
  • Civility - Treat every user with respect; do not insult them or harass them. Doing this will result in a warning and if continued, the user will be blocked from editing.
  • Images - All images must have proper and understandable names.
  • Badges Boosting - All users must remember to edit properly and should not make poor/bad faith edits to get badges.
